## Install

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/Robin831/Forge/main/install.sh | bash
```

**macOS (Homebrew):**
```bash
brew install Robin831/forge/forge
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/Robin831/Forge/main/install.ps1 | iex
```

**Windows (Scoop):**
```
scoop bucket add forge https://github.com/Robin831/scoop-forge
scoop install forge
```

Or download the archive for your platform below.

## Features

- **Hearth PR panel**: dedicated overlay showing all open PRs across anvils with Merge, Fix CI, Fix Comments, and Fix Conflict actions (Forge-7w6r, Forge-5vb3)
- **Anvil health badges**: Queue panel shows per-anvil health indicators and poll events are filtered for cleaner event log (Forge-uqb4)
- **Queue stop**: `forge queue stop` fully stops bead processing for a given bead (Forge-66ef)
- **Workers panel**: wider layout (40% of screen), uses Bubbles table component (Forge-e50w)
- **Log viewer**: press 'o' in Workers panel to view Smith logs (Forge-x1bs)
- **External PR monitoring**: non-Forge PRs in registered anvils are now tracked and actionable
- UPX binary compression for smaller downloads (Forge-q72n)
- SBOM generation for supply chain security (Forge-ztgx)

## Bug Fixes

- **Bellows**: external PRs (not created by Forge) no longer trigger cifix/reviewfix/rebase workers — they are tracked for display in the PR panel only (Forge-4h4i)
- **Retry reset**: now resets bead status from in_progress back to open so the poller can re-dispatch retried beads and paused crucibles (Forge-4zea)
- Fix worktree branch reset to base ref on pipeline retry (Forge-h73z)
- Fix Workers panel border misalignment caused by table cell padding
- Fix dispatcher race condition with atomic.Pointer and hot-reload support
- Fix auto-learn distillation errors (Forge-k0nj)
- Improve PR descriptions with detailed change summaries (Forge-jhif)
