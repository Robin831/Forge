## Install

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/Robin831/Forge/main/install.sh | bash
```

**macOS (Homebrew):**
```bash
brew install Robin831/forge/forge
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/Robin831/Forge/main/install.ps1 | iex
```

**Windows (Scoop):**
```
scoop bucket add forge https://github.com/Robin831/scoop-forge
scoop install forge
```

Or download the archive for your platform below.
