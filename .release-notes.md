## Install

**Linux / macOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/Robin831/Forge/main/install.sh | bash
```

**macOS (Homebrew):**
```bash
brew install Robin831/forge/forge
```

**Windows (PowerShell):**
```powershell
irm https://raw.githubusercontent.com/Robin831/Forge/main/install.ps1 | iex
```

**Windows (Scoop):**
```
scoop bucket add forge https://github.com/Robin831/scoop-forge
scoop install forge
```

Or download the archive for your platform below.

## Features

- **Hearth PR panel**: dedicated overlay showing all open PRs across anvils with Merge, Fix CI, Fix Comments, and Fix Conflict actions
- **Anvil health badges**: Queue panel shows per-anvil health indicators and poll events are filtered for cleaner event log
- **Queue stop**: `forge queue stop` fully stops bead processing for a given bead
- **Workers panel**: wider layout (40% of screen), uses Bubbles table component
- **Log viewer**: press 'o' in Workers panel to view Smith logs
- **External PR monitoring**: non-Forge PRs in registered anvils are now tracked and actionable
- UPX binary compression for smaller downloads
- SBOM generation for supply chain security
