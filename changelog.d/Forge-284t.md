category: Added
- **Release notifications via webhooks** - New `forge notify release` command sends release announcements to configured webhooks when a new Forge version is published. Teams channels receive a rich Adaptive Card; custom endpoints (dashboards, Slack, etc.) receive a generic JSON payload. The release GitHub Actions workflow now also curls configured webhook URLs from `RELEASE_WEBHOOK_URL` and `TEAMS_RELEASE_WEBHOOK_URL` secrets automatically on tag push. Configure additional generic webhook URLs via `notifications.release_webhook_urls` in `forge.yaml`. (Forge-284t)
